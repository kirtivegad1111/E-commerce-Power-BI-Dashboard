# E-commerce-Power-BI-Dashboard
This project analyzes an e-commerce dataset using **Power BI**.   The dashboard provides insights into revenue, customer distribution, product performance, and order trends.
## 🛠️ Tools & Skills Used
- **SQL** (data preparation)
- **Power BI Desktop** (data modeling & visualization)
- **Data Analysis** (KPIs, trends, customer segmentation)

## 📈 Dashboard Highlights
1. **Total Revenue Card**  
   - Shows combined revenue from product price and freight value.

2. **Top Cities by Customers**  
   - Bar chart showing customer distribution across cities.  
   - São Paulo, Rio de Janeiro, and Belo Horizonte are top contributors.

3. **Top Products by Revenue**  
   - Bar chart showing which products generate the most revenue.

4. **Monthly Orders Trend**  
   - Line chart showing order volume across months.  
   - Helps identify seasonal peaks and dips.
5. **Interactive Slicer**  
   - Filter dashboard by customer city for deeper insights.

## 🚀 Insights
- São Paulo has the highest customer base.  
- Certain products dominate revenue contribution.  
- Orders show clear monthly fluctuations, useful for planning logistics and marketing.
